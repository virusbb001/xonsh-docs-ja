class xonsh.pyghooks.XonshStyle(style_name='default')[ソース]¶

A xonsh pygments style that will dispatch to the correct color map by using a ChainMap. The style_name property may be used to reset the current style.

Parameters
style_namestr, optional

The style name to initialize with.

Enhance colors when using cmd.exe on windows. When using the default style all blue and dark red colors are changed to CYAN and intense red.
