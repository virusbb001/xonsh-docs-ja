JSON Utilities (xonsh.jsonutils)¶

Custom tools for managing JSON serialization / deserialization of xonsh objects.

xonsh.jsonutils.serialize_xonsh_json(val)[ソース]¶

JSON serializer for xonsh custom data structures. This is only called when another normal JSON types are not found.
