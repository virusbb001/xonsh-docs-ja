Compiling and Caching of Xonsh Code (xonsh.codecache)¶

Tools for caching xonsh code.

xonsh.codecache.code_cache_check(cachefname)[ソース]¶

Check whether the code cache for a particular piece of code is valid.

Returns a tuple containing: a boolean representing whether the cached code should be used, and the cached code (or None if the cache should not be used).

xonsh.codecache.code_cache_name(code)[ソース]¶

Return an appropriate spoofed filename for the given code.

xonsh.codecache.compile_code(filename, code, execer, glb, loc, mode)[ソース]¶

Wrapper for execer.compile to compile the given code

xonsh.codecache.get_cache_filename(fname, code=True)[ソース]¶

Return the filename of the cache for the given filename.

Cache filenames are similar to those used by the Mercurial DVCS for its internal store.

The code switch should be true if we should use the code store rather than the script store.

xonsh.codecache.run_code_with_cache(code, execer, glb=None, loc=None, mode='exec')[ソース]¶

Run a piece of code, using a cached version if it exists, and updating the cache as necessary.

xonsh.codecache.run_compiled_code(code, glb, loc, mode)[ソース]¶

Helper to run code in a given mode and context

xonsh.codecache.run_script_with_cache(filename, execer, glb=None, loc=None, mode='exec')[ソース]¶

Run a script, using a cached version if it exists (and the source has not changed), and updating the cache as necessary.

xonsh.codecache.script_cache_check(filename, cachefname)[ソース]¶

Check whether the script cache for a particular file is valid.

Returns a tuple containing: a boolean representing whether the cached code should be used, and the cached code (or None if the cache should not be used).

xonsh.codecache.should_use_cache(execer, mode)[ソース]¶

Return True if caching has been enabled for this mode (through command line flags or environment variables)

xonsh.codecache.update_cache(ccode, cache_file_name)[ソース]¶

Update the cache at cache_file_name to contain the compiled code represented by ccode.
