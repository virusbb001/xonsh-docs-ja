Parser (xonsh.parser)¶

Implements the xonsh parser.

class xonsh.parser.Parser(lexer_optimize=True, lexer_table='xonsh.lexer_table', yacc_optimize=True, yacc_table='xonsh.parser_table', yacc_debug=False, outputdir=None)[ソース]¶

A Python v3.6 compliant parser for the xonsh language.

Parameters
lexer_optimizebool, optional

Set to false when unstable and true when lexer is stable.

lexer_tablestr, optional

Lexer module used when optimized.

yacc_optimizebool, optional

Set to false when unstable and true when parser is stable.

yacc_tablestr, optional

Parser module used when optimized.

yacc_debugdebug, optional

Dumps extra debug info.

outputdirstr or None, optional

The directory to place generated tables within.
